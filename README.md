### ⭐ [Full-Stack OAuth2 Cloud Application](https://github.com/oveshshaikh786/fullstack-oauth2-app)

A production-grade full-stack application with secure OAuth2 authentication, containerized deployment, and cloud infrastructure on AWS.

`Spring Boot 3` `React 18` `Spring Security` `PostgreSQL` `Docker` `AWS EC2 / RDS` `GitHub Actions`

- Built a Spring Boot 3 REST API backend with a React 18 frontend, documented end-to-end via Swagger / OpenAPI
- Implemented OAuth2 social login (Google & GitHub) using Spring Security, issuing JWT tokens for stateless session management
- Managed database schema and migrations with Flyway on PostgreSQL; ran all services locally via Docker Compose
- Deployed containerised backend and frontend to AWS EC2 with a PostgreSQL RDS instance; automated image builds with GitHub Actions
- Configured HTTPS with a Let's Encrypt SSL certificate for a secure, production-ready deployment

Java OOP project modelling a zoo with Tiger, Dolphin, and Penguin — with a Swing desktop UI and a live web version.

`Java` `Swing` `OOP` `Interfaces` `Serialization` `HTML/CSS/JS` `GitHub Pages`

- Designed with an abstract `Animal` base class and `Walk`, `Swim`, `Eat` interfaces — each animal implements only what it can do
- Identified and fixed 6 bugs across 5 source files (missing validation, redundant interface declarations, broken `toString()`, placeholder parsing bug in Swing UI)
